Filter and replace
  • FILTER ~

We have 3 ways to filter :
*suppose our task is to remove all the numbers from 5 to 7 inclusive

List<Integer> nums = new ArrayList<>(Arrays.asList(1,3,5,7,8,9,10));

//1.old school method:
Iterator<Integer> iterator = nums.iterator();
while (iterator.hasNext()){
    int a = iterator.next();
    if (a>=5&& a<=7)
        iterator.remove();
}

//2.use new method with lambda expr.
nums.removeIf(a -> a >= 5 && a <= 7);

//3.using Stream API
nums = nums.stream().filter(n-> (n>0 && n<5)|| n>7).collect(Collectors.toList());
*1 and 2 modifyÂ the actual collection , 3 - we filter the stream received from the collection

  • REPLACE ~

*for exaple ,we need to square each element ([0,1,2,3] -> [0,1,4,9])

//new method of replacing
nums.replaceAll(n -> n*n);

//replace all strings starts with "a" to "str"
strs.replaceAll( a-> {
    if (a.startsWith("a")) return "str";
    else return a;
});

//using Stream API(it made at least some sense to add more operations)
nums.stream().map(n -> n*n).filter(n -> n>0).forEach(System.out::println);//output: 1 4 9
