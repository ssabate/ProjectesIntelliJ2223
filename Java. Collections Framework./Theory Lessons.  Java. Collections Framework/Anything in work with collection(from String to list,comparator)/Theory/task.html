How to turn a string into a collection?

the answer is very simple...
For example your BufferedReader read string as this "1 2 3 4 5"
you have several ways:

BufferedReader reader = ...;
String s = reader.readLine();//its "1 2 3 4 5"
List<Integer> list = new ArrayList<>();
//use Stream API
List<Integer> arrayList = Arrays.stream(s.split("\\s")).mapToInt(Integer::valueOf).boxed().collect(Collectors.toList());
*"\\s" = its space or "  "
*result of s.split("\\s") = String[]
