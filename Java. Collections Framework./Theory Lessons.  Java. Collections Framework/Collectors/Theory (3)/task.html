Collectors that produce values

We know one of this methods of Collectors from previous lesson : averagingLong()
This method have several similar methods :

  • averagingInt, averagingLong, averagingDouble;

    as well as these methods, whose names speak for themselves:
  • summingInt, summingLong, summingDouble;
  • maxBy, minBy
  • counting

    For example I can show you the following code with short comments:
    This code calculates the sum of ages of all persons over 18
Integer agesSumOfPersonsOlderThan18 = persons.stream().filter(p -> p.age>18).collect(Collectors.summingInt(p->p.age));
But this is only the first way! Here is the second:
Integer agesSumOfPersonsOlderThan18 = persons.stream().filter(p -> p.age > 18).mapToInt(p -> p.age).sum();

The situation is similar with other methods
max() and maxBy() , counting() and count()...
